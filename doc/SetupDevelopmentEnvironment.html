LINKS AND TIPS

Development environment:
  • AI2 : http://ai2.appinventor.mit.edu/ (Google account required)



    At mobile settings, allow Unknown Sources ( others than from Play Store).
Source code dependencies:
  • JSCH : https://github.com/is/jsch/tree/mast...om/jcraft/jsch
  • JZLIB : https://github.com/ymnk/jzlib/tree/m...m/jcraft/jzlib


Binary project dependencies:
  • JSCH : https://sourceforge.net/projects/jsc...ch.jar/0.1.54/ (note, it is the above source code compiled)



    • At Environment Variables, add to CLASSPATH: 
      <main folder>\appinventor\components\runtime\jcraft\jsch \jgss
    • At \appinventor\components\ locate the file Build.xml and seek for the comment header CopyComponentLibraries and add the following sentence: 
      <copy toFile="${public.deps.dir}/jsch-0.1.54.jar" file="${lib.dir}/jsch/jsch-0.1.54.jar" />
Required/Preferred tools:
  • Apache Ant 1.10.6 : https://ant.apache.org/bindownload.c...et/pub/apache/
  • Java SDK 8 : https://www.oracle.com/technetwork/j...s-2133151.html
  • Git terminal 2.2.1.0 : https://git-scm.com/download/
  • Code editor : https://notepad-plus-plus.org/download/v7.6.6.html
  • Logcat Standalone : https://developer.android.com/studio...orm-tools.html



    • If you have Android Studio, add to PATH Environment Variables:%HOME_USER%\AppData\Local\Android\Sdk\platform-tools
    • At mobile settings:



      • Enable USB Debug,
      • Turn mobile into Development Mode.
